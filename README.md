# Mood Tracker 

A modern and responsive mood tracking application built with **HTML**, **CSS**, and **JavaScript**.

Track your emotions, save your daily mood, and monitor your emotional history through a clean and intuitive user interface.

---

## 🚀 Features

* Add daily mood entries
* Multiple mood selections
* Mood history tracking
* Local Storage support
* Automatic data saving
* Responsive design
* Modern glassmorphism UI
* Daily mood statistics
* Clean and user-friendly interface
